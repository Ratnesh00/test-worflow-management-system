# Task Workflow Management System
An application built using Java and Spring Boot to manage task assignment, workflow approvals, and status tracking within an organization.
The system supports role-based access for Admin, Manager, and Employee users, enabling secure task creation, assignment, progress updates, and audit logging.
This project also demonstrates modern DevOps practices including Docker containerization, CI/CD pipeline setup using GitHub Actions, and automated cloud deployment on Render.

## Tech Stack
- JDK 21
- Maven 3.9.15
- IntelliJ 2024
- MySQL Server 8.0
- MySQL Workbench 8.0 CE
- Git
- Postman
- Docker Desktop
- GitHub
- Render

## Features
- User registration and login with JWT authentication
- Role-based access control (Admin / Manager / Employee)
- Task creation, assignment, and status tracking
- Priority-based workflow management
- Task comments and audit logs
- RESTful APIs with validation
- MySQL database integration
- Dockerized deployment
- CI/CD pipeline with GitHub Actions
- Auto deployment on Render
